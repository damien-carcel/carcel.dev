# carcel.dev

## Requirements

- Docker
- Docker Compose v2
- Task

## How to use it?

First, install the dependencies with `task dependencies:install`.

Then run the application by using either:
- `task run:dev` for development mode with hot-reload and access it through [http://localhost:8000](http://localhost:3000).
- `task run:prod` for production-like mode and access it through [http://localhost:8000](http://localhost:8000).

The full list of actions can be accessed by running the following command in your usual terminal:
```bash
$ task
```
