# carcel.dev

## Requirements

- Docker
- Docker Compose v2
- Task

## How to use it?

The full list of actions can be accessed by running the following command in your usual terminal:

```bash
$ task
```

Optionally, if using VS Code, accept to open the project in Dev Containers when prompted.
